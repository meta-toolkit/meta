template<class Similarity>
class meta::clustering::basic_single_link_policy< Similarity >

A simple linking policy for agglomerative clustering utilizing the single-link metric.

The single link metric is one that merges cluster \(X\) and cluster \(Y\) which have the minimum distance \(D_{xy}\) between them, where \(D_{xy}\) is defined as being the minimum distance between any point in cluster \(X\) and any other point in cluster \(Y\).
